Arduino-mood-light

arduino moodlight with selectable modes

View project onGitHub

An Arduino powered moodlight. Download the design files for a laser cut 1/4" plywood base and housing at Thinigiverse

Moodlight Modes: 1. Random Fading 2. Random Color 3. Rainbow 4. Candle 5. User Controlled 6. Full Spectrum Rainbow (HSV) 7. Off
